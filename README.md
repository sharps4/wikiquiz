# SharpsAI

This project is a little game based on the Wikipedia API, where the objectives is to guess the article title.

## Installation

```bash
git clone https://github.com/sharps4/wikiquiz
npm install
```

## Usage

Start the server :

```bash
cd server
npm start
```

Start the react-app

```bash
cd wikiquiz
npm start
```
