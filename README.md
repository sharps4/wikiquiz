# SharpsAI

The site presents you with a random article with the title hidden and the goal is to find it only with the information present in the article.

## Installation

```bash
git clone https://github.com/sharps4/wikiquiz
npm install
```

## Usage

Start the server :

```bash
cd server
npm start
```

Start the react-app

```bash
cd wikiquiz
npm start
```
